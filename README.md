# Coffee Shop Web Application

This is a web application developed using Flask.

## Technologies Used
- HTML
- CSS
- JavaScript
- Python Flask
- SQLite Database

## Features
- User registration form
- Form validation using JavaScript
- Feedback storage in database
- Responsive design
